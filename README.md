## Run Locally

Clone the project

```bash
  git clone https://github.com/pranahonk/adsi-logical-test
```

Go to the project directory

```bash
  cd adsi-logical-test
```

Dependency Install the project

```bash
  yarn install || yarn || npm install
```

Run Locally
```bash
  yarn serve
```


Open Browser and type 
```bash
  http://localhost:8080/
```
